description: Push the current state of the portfolio to GitHub
---

This workflow will add all changes, commit them, and push to the configured remote repository.

1. Add all changes to the staging area
// turbo
```bash
git add .
```

2. Commit the changes
```bash
git commit -m "Update portfolio content and theme"
```

3. Push to the remote repository
```bash
git push origin master
```
